Purpose:
Comp 3005 final project that simulates a health & fitness club management system
3 types of users with different operations each
connected to a postgresql database 

Video Demo:
https://youtu.be/fzKifNSt9Ds

Launch instructions:

First create a postgresql database in a program of your choice

To initialize the program, first create the 3 types of users: Members, Trainers, Admins.
Then execute the the rest of the DDL to create all other tables.

Make sure the database is running and accessable.

Execute the DML next.

In main.py, change the password and name of a postgresql database to fit your own on line 6 and 7.
You might need to change the host, user, and/or port number on line 9.

Launch main.py.

Follow the Video Demo.
